# Front & Back Matter Generator
A modern web-based tool to quickly generate professional **Front Matter** and **Back Matter** pages for books, ebooks, PDFs, and publishing projects. Built with a clean UI, dark/light mode support, responsive layout, and export-ready formatting.

## ✨ Features

* 📘 Generate Front Matter pages
* 📙 Generate Back Matter pages
* 🌙 Dark / Light Theme Toggle
* 📱 Fully Responsive Design
* 🎨 Modern UI with Outfit Font
* 📋 Copy Output with One Click
* ⚡ Fast and Lightweight
* 🧩 Clean HTML, CSS & JavaScript Structure

## 🚀 Technologies Used
* HTML5
* CSS3
* JavaScript
* Bootstrap
* Google Fonts (Outfit)

## 🖥️ Installation
Clone the repository:

```bash
git clone https://github.com/sufyanism/Front_Back_Matter_Generator.git
```

Open the project folder:
```bash
cd Front_Back_Matter_Generator
```
Run `index.html` in your browser.

## 🎯 Usage
1. Enter your required content
2. Generate Front or Back Matter
3. Copy the generated output
4. Use in your publishing workflow
